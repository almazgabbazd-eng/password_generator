# Random Password Generator

**Автор:** Алмаз Габбазов

## Описание

Приложение для генерации случайных паролей с настраиваемыми параметрами и сохранением истории.

## Функциональность

- Генерация паролей заданной длины (4–64 символа)
- Выбор типов символов: буквы, цифры, спецсимволы
- Копирование пароля в буфер обмена
- Сохранение истории последних 20 паролей
- Отображение истории в табличном виде

## Требования

- Python 3.6+
- Библиотеки:
  - `tkinter` (встроен)
  - `pyperclip` для копирования в буфер обмена
